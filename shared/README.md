# VGfriend Shared Types

Shared TypeScript types for VGfriend monorepo.

## Usage

```typescript
import { User, Character, Message } from '@vgfriend/shared';
```

## Building

```bash
npm run build
```

## Available Types

- `User` - User model
- `Character` - AI character model
- `Message` - Chat message
- `Quest` - Quest/mission
- `Gift` - Shop item
- `Scene` - Chat scene/background
- `Memory` - User memory/milestone
- And more...
